# -----------------------------------------------------------------------------
# Framework principal
# -----------------------------------------------------------------------------

# FastAPI: framework web async de alto rendimiento.

# Uvicorn: servidor ASGI para correr FastAPI.

# Starlette: base de FastAPI, necesaria para middlewares y routing.

# -----------------------------------------------------------------------------

# Alembic: migraciones de base de datos.
